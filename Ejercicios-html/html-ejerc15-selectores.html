Selector de Clase

Este párrafo usa un selector de clase para cambiar su color y hacerlo en negrita

Selector de Identificador

Este párrafo usa un selector de identificador para cambiar su color a rojo y aumentar el tamaño del texto

Selector Universal

El selector universal aplica la fuente arial a todos los elementos de la página
